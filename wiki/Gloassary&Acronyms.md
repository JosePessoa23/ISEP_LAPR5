# Glossary and Acronyms
ML - Módulo de Logística.
MGA - Módulo de Gestão de Armazéns.
SPA - Módulo Interface Utilizador.
CPD - Centro de Processamento de Dados.
MTD - máximo tempo tolerável de inoperacionalidade.
MTPD - tempo máximo de disrupção.
EletricGo - Nome do sistema.
Plano de entregas - planeamento das entregas necessárias aos vários armazéns/lojas tendo por base as previsões de vendas e encomendas efetuadas.
Administrador de sistema – gere os utilizadores e autorizações dos mesmos.
Gestor de Armazém– gere os dados dos armazéns e entregas.
Gestor de frota – gere os dados dos camiões.
Gestor de Logística – gere os dados dos percursos e planeamento da distribuição.
